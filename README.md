# Documentación de SimPy con ejemplos prácticos

Este repositorio contiene una documentación introductoria y ejemplos prácticos de la librería **SimPy**, un framework de simulación de eventos discretos basado en procesos para Python [citation:1][citation:5].

## Contenido

- **[¿Qué es SimPy?](que-es-simpy.md)**: Una descripción general de la librería, sus conceptos fundamentales y casos de uso.
- **Ejemplos de Simulación**: Tres ejemplos prácticos que demuestran diferentes capacidades de SimPy:
    - [Ejemplo 1: Simulación de un Banco](ejemplo-banco.md)
    - [Ejemplo 2: Simulación de un Taller Mecánico](ejemplo-taller-mecanico.md)
    - [Ejemplo 3: Simulación de una Cafetería](ejemplo-cafeteria.md)
